# Student Placement Prediction

This is a simple machine learning project where I used Logistic Regression to predict whether a student will get placed or not based on their CGPA and IQ score.

## What I did in this project

* Loaded the dataset using Pandas
* Cleaned and explored the data
* Visualized the data using Matplotlib
* Split the data into training and testing sets
* Applied StandardScaler for feature scaling
* Trained a Logistic Regression model
* Evaluated the model using accuracy score
* Predicted placement for new student data

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Result

The model achieved around **80% accuracy** on the test data.
